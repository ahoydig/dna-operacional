# Font Pairings

Max 3 pesos por familia. Nunca usar combos overused.

## Combos Overused (PROIBIDOS)
- Montserrat + Open Sans
- Poppins + Roboto
- Inter + Inter
- Playfair Display heading + Inter body
- DM Serif Display heading + DM Sans body

## Serif + Sans (Editorial)
- Playfair Display + Source Sans Pro
- Libre Caslon Display + Libre Franklin
- Merriweather + Merriweather Sans
- Fraunces + DM Sans

## Sans + Sans (Modern/Tech)
- Clash Display + Clash Grotesk
- Sora + Geist
- Supreme + Inter (ok como fallback)
- Mona Sans + Hubot Sans

## Mono + Sans (Dev/Tech)
- Geist Mono + Geist
- IBM Plex Mono + IBM Plex Sans
- Bergen Mono + JUST Sans (projeto MaestrIA)

## Condensed + Regular (Editorial)
- Roboto Condensed + Roboto
- Barlow Condensed + Barlow

## Display Impact
- Clash Display + Inter
- Unbounded + Geist
- Syne + Inter
- Costly + JUST Sans (projeto MaestrIA)

## Premium (Envato Elements)
Preferir fontes premium sobre Google Fonts para diferenciacao.
Categorias recomendadas: Display bold, Sans clean, Monospace.
