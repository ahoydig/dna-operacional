**DNA Operacional v0.1.0** — 14 skills consolidadas num plugin instalável. Tudo que eu uso pra criar conteúdo, tocar agência e mapear concorrência.

**Install (2 comandos):**

```
/plugin marketplace add ahoydig/dna-operacional
/plugin install dna-operacional
```

Depois digita `/dna` pra ver o menu. `/setup-projeto` te guia do zero.

**O que tem dentro:**

- 🎯 Setup + voz (configura projeto + cria voz da marca)
- 🔍 Pesquisa diária BR + concorrentes Instagram + raio-x de ads
- 🎬 Ciclo completo de conteúdo (ideias → roteiro → carrossel → análise)
- 🤖 Humanizer que lê voz dinâmica + auto-melhoria

**Como começar:**

1. Instala o plugin (2 comandos acima)
2. Configura APIs externas (guia em `docs/APIS-EXTERNAS.md` — zero chave pessoal minha, cada um configura na sua conta)
3. `/setup-projeto` no primeiro projeto
4. Segue os "Próximos Passos" que cada skill te mostra

Repo: https://github.com/ahoydig/dna-operacional

Bugs ou sugestões: abre issue no GitHub. Próximo release (v0.2.0) trás skills de agência — onboarding-cliente, família meta-ads, coletar-anuncios.
